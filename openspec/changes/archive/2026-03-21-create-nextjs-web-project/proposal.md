## Why

This project needs a modern web frontend to provide a user-friendly interface. Setting up a Next.js project with TypeScript, Tailwind CSS, shadcn/ui, and the App Router will establish a solid foundation for building a scalable, type-safe, and beautifully designed web application with excellent developer experience.

## What Changes

- Initialize a new Next.js project in the `/web` directory
- Configure TypeScript for type safety and better developer experience
- Set up Tailwind CSS for utility-first styling
- Install and configure shadcn/ui component library
- Enable Next.js App Router for modern routing and server components
- Create initial project structure with common directories (app, components, lib, etc.)
